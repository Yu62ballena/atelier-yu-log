# design.md — Atelier Yu Log ダッシュボード

対象：`dashboard/` ディレクトリ（Issue 3）のみ。collector・pipelineには適用しない。

## コンセプト

- 利用者は開発者本人1人のみ。**見た目の華やかさより、情報を一瞥で把握できる密度**を優先する。
- 毎日ちらっと開いて「今日は何にどれだけ使ったか」を数秒で確認できる画面にする。
- 派手な装飾・アニメーションは不要。データそのものが主役。

## カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| 背景（ベース） | `#FAFAF9` | ページ全体の背景 |
| 背景（カード） | `#FFFFFF` | カード・パネルの背景 |

ダークモードは初回リリースでは不要（未決定事項に記載、後回し）。

## タイポグラフィ

- フォントファミリー：システムフォント（`-apple-system, BlinkMacSystemFont, "Hiragino Sans", sans-serif`）
- 見出し（画面タイトル）：20px / bold
- カードタイトル：14px / semibold / テキスト（副）色
- 数値（合計時間など）：28px / bold ← ダッシュボードの主役なので大きく
- 本文・ラベル：13px / regular

## レイアウト方針

- サイドバー幅：200px 固定。ラベルのみ（アイコン任意）。5画面（今日／期間集計／階層ビュー／日報一覧／設定）をリスト表示。
- メインエリア：最大幅1200px、中央寄せ、左右パディング32px。
- カード間の余白：16px。カード内パディング：20px。角丸：8px。影は使わない（`border`のみで区切る）。

## UIコンポーネント方針

- **タブ切替**（期間集計の日/週/月、軸切替）：下線タイプ。選択中はアクセント色の下線＋テキストをアクセント色に。
- **ボタン**：最小限（設定画面の保存ボタンのみ想定）。背景アクセント色、白文字、角丸6px、パディング上下8px・左右16px。
- **タイムライン**（今日ビュー）：横棒1本、時間軸0:00〜24:00を等幅で表現。プロジェクト色で色分け、AFKはグレー。ホバーでツールチップ（時間帯・アプリ名・タイトル）。
- **サンバースト**：Rechartsで直接サポートされないため、`d3`を用いて実装してよい（既定ライブラリに追加可）。クリックで中心を再描画するドリルダウン必須。

## アニメーション方針

- 画面遷移・タブ切替：アニメーションなし（即時切替）。
- チャートの初回描画のみ：0.3秒のフェードイン。
- それ以外の凝ったアニメーションは付けない（個人用ツールのため実装コストに見合わない）。
